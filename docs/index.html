The Workflow

How LoreAI scales knowledge

01

Distill

Lore captures file structures, past bugs, and architectural choices, compressing them into a high-value memory layer.

02

Store

Your data stays local in a specialized SQLite database (lore.db), ensuring privacy and lightning-fast retrieval.

03

Contextualize

When you ask your AI to code, Lore injects the exact project "wisdom" needed to avoid hallucinations and repeat errors.

Core Tech

Operational Intelligence

Lore doesn't just see your code; it remembers your *intent*. It tracks why decisions were made, making your AI a true partner. It tracks file paths, error messages, and exact commands.

Architecture

Three-Tier Memory

Instead of basic summarization, Lore uses a sophisticated distillation process to categorize knowledge, keeping context relevant and precise.

Integration

Plug & Play with Claude Code

Get started in seconds. LoreAI integrates directly with your existing AI stack (Claude Code, Pi, OpenCode) via a simple JSON configuration.

// opencode.json
{
  "plugin": [
    "@loreai/opencode"
  ]
}
