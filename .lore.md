* **Always preserve raw bytes when handling HTTP request bodies**: Handle HTTP bodies as Buffer\[] with Buffer.concat, never string concatenation. Use latin1 encoding for text search in binary bodies. Copy bytes before mutation; guard offset writes. Pass as bytes not strings.

<!-- lore:019ec82b-5e87-7913-8d9f-6a8c34d6ea81 -->
