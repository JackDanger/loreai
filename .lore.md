### Pattern

* **Auto-import guard: last\_import\_at timestamp replaces sentinel \_\_declined\_\_ rows**: In \`packages/core/src/import/\`, the \`maybeAutoImport()\` guard uses \`last\_import\_at\` on the project record. \`recordOffered(projectPath)\` sets this timestamp unconditionally after the prompt fires (before user answer). \`wasOffered()\` checks \`last\_import\_at IS NOT NULL\`. <!-- lore:019e216f-0d76-71dc-b3e6-74bec2088c86 -->
* **Bun --compile WASM embedding: $bunfs asset paths for onnxruntime-web**: To embed onnxruntime-web WASM files in a Bun \`--compile\` binary: (1) wrapper imports WASM files as \`{ type: 'file' }\` assets and sets \`globalThis.\_\_LORE\_VENDOR\_WASM\_PATHS\_\_\` (object form \`{ mjs, wasm }\` with exact hashed \`$bunfs\` filenames); (2) \`binaryExternalsPlugin\`'s \`onLoad\` hook patches transformers.js at esbuild time to read \`wasmPaths\` from \`globalThis.\_\_LORE\_VENDOR\_WASM\_PATHS\_\_\` instead of CDN URL.
